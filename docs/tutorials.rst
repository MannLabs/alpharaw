Tutorials of AlphaRaw
===========================

AlphaRaw is designed to support RAW data access from three main
MS-proteomics vendors: `thermo`, `sciex`, and `bruker`.
To visually inspect bruker data, please use our
`AlphaTims <https://github.com/mannlabs/alphatims>`_ package (and see the tutorials therein).

Also, community formats like `mzml` are supported.
