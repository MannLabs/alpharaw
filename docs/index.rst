Documentation for AlphaRaw
=====================================

AlphaRaw is a pure Python package which is designed to load MS data of different formats
(Thermo, Sciex, mzml, ...) into a standard DataFrame (tabular) format.
For more information, see AlphaRaw on `GitHub <https://github.com/MannLabs/alpharaw/>`_.
